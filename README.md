---

## ⚙️ Core Architecture & Features

This library intercepts the automation queue to maintain autonomous productivity with zero manual intervention:

1. **Intelligent Polling:** Continuously checks the backend proxy APIs for high-priority task queues securely.
2. **Multi-LLM Dispatcher:** Efficiently splits rendering tasks to Gemini 2.0 or DeepSeek based on system load, preserving your rate limits.
3. **Resilient Retry Protocol:** Employs exponential backoff capabilities via the `.agent/skills/error-handler` matrix ensuring tasks survive transient API outages.
4. **VLM Validation Checks:** Offloads image screening to headless visual models (like Qwen VLM) to ensure *cinematic food imagery* or exact Amazon product representation without fail.
5. **Headless Sync:** Dispatches compiled JSON and rich styling formats directly to your CMS via protected REST endpoints, verifying payload integrity across the wire.
